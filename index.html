Objetivo de las secciones

Brindar conocimiento a los interesados a cerca de la implementación de la teoría general de sistemas en las organizaciones, como una fuente ágil de asumir cada proceso. Primero que todo se debe entender que es un sistema, dicho esto se endiente que es una reunión de partes que interactúan entre sí. De otro lado como bien se sabe una organización puede ser una empresa o una determinada entidad. De tal forma lo que se quiere es expresar cómo funcionan mancomunadamente estos dos factores y cuál es la incidencia tanto del uno como del otro, en el progreso que buscan las personas dirigentes de los entes mencionados. Para que un sistema funcione este depende de todas las partes que lo conforma de tal forma que cada una de ellas case, y de tal forma hagan el normal funcionamiento del sistema. En una organización es sistema es igual, todo tiene que tener un proceso y cada proceso una secuencia, de tal forma que cada uno de los procesos cumplan a cabalidad con el objetivo esperado. De tal forma lo dicho anteriormente se define como: insumos procesos y resultados. A diario estamos empleando una serie de procesos en cada una de las actividades de nuestro diario vivir, estos sufren una transformación para dar lugar a un resultado. Hay sistemas abiertos como también hay sistemas cerrados. Un ejemplo de un sistema abierto es: un restaurante, porque hay elementos que entran y salen es decir, al sistema entran elementos y sale producción es decir esta en interacción con el medio, de otro lado si es lo contrario quiere decir que es un sistema de carácter cerrado, por citar un ejemplo para este último es por ejemplo una isla es un sistema cerrado, ya que no hay interacción con otros entes.
